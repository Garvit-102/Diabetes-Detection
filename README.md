🩺 Diabetes Detection using Machine Learning

An end-to-end Machine Learning project that predicts whether a person is diabetic or not based on medical diagnostic features. This project demonstrates data preprocessing, model training, evaluation, and deployment-ready structure following MLOps best practices.

📌 Project Overview

Diabetes is a chronic disease that affects millions of people worldwide. Early detection can help in timely treatment and prevention of complications.

This project builds a classification model using medical attributes such as:

Pregnancies

Glucose

Blood Pressure

Skin Thickness

Insulin

BMI

Diabetes Pedigree Function

Age

The model predicts:

0 → Non-Diabetic
